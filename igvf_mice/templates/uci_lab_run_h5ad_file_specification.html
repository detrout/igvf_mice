Contents of the AnnData h5ad file{% if filename %} {{ filename }}{% endif %}

The AnnData (annotated data, or adata) file {% if filename %} {{ filename }}{% endif %} contains detailed sample metadata as well as cell type annotations in the observation (obs) table and gene information in the variables table (var). Unique cell IDs are located in adata.obs.index. The AnnData file contains both raw and CellBender denoised gene counts matrices in separate layers: adata.layers['raw_counts'] and adata.layers['cellbender_counts']. Cells or nuclei are always filtered by >0.5 cell_probability from CellBender analysis along with additional QC filters which are indicated in the attached documents.

After filtering, data for each tissue was normalized and clustered using CellBender denoised counts. Normalization includes regression of technical parameters (number of genes expressed per cell and percent mitochondrial gene expression). The current adata.X cell-by-gene counts matrix contains the normalized data. {% if harmony %}Harmony was used to integrate exome capture and non-exome capture subpools for annotation. {% endif %} A Leiden clustering resolution of 1 was used to define clusters. Cell type annotations were assigned per Leiden cluster or subcluster (leiden_R) using expression of canonical cell type marker genes.
